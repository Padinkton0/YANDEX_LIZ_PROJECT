# Digital Navigator

**DigitalNavigator** — это веб-сайт, интерактивная платформа, где пользователи через пошаговые симуляции мобильных приложений осваивают базовые и продвинутые навыки цифровой грамотности. Обучение построено на практических сценариях (основы смарфона, мессенджер MAX, госуслуги, онлайн‑покупки в приложении Пятерочка) и проходит в виде кликабельных экранов с подсказками и голосовыми инструкциями.

Каждый тренинг разбит на логические этапы: демонстрация — выполнение — проверка. Результаты сохраняются в личном кабинете, есть два уровня сложности и возможность повторного прохождения для закрепления навыков. Все симуляции безопасны — без доступа к реальным аккаунтам и персональным данным, интерфейс адаптирован для новичков и пожилых людей.

Ключевые преимущества: симуляции экранов, пошаговый формат, озвучка с учётом политики браузеров, учёт прогресса, простота и доступность.

## Возможности проекта

- регистрация и авторизация пользователей
- выбор уровня обучения (базовый / продвинутый)
- прохождение обучающих модулей:
  - основы смартфона
  - работа с мессенджером
  - портал Госуслуг
  - онлайн-покупки
- отслеживание прогресса и достижений в личном кабинете
- Telegram-бот для:
  - помощи пользователям
  - отправки вопросов специалисту и получения обратной связи
---

## Используемые технологии

- **Python 3**
- **Flask** — веб-приложение
- **Flask-Login** — аутентификация пользователей
- **Flask-WTF** — интеграция WTForms с Flask
- **SQLite** — база данных
- **SQLAlchemy** — ORM для работы с базой данных
- **Werkzeug** — утилиты и безопасность (хеширование паролей, middleware)
- **HTML / CSS / Bootstrap** — интерфейс
- **aiogram** — Telegram-бот
- **Render.com** — хостинг
- **JavaScript** — интерактивные элементы и тренинги
---

## Установка и запуск проекта

1. Клонировать проект с помощью ссылки в репозитории, либо скачать zip-файл с проектом и распаковать проект.
2. Установить requirements: pip install -r requirements.txt
3. Запустить проект с помощью run.py
